CPSC 314 Final Project Presentation

Book Finder

Elizabeth Larson

Project Overview/Introduction

  • COVID-19 Pandemic
    • Lack of access to books at home
    • Remote learning and closed public librares
  • This Website
    • Compilation of books that deal with serious/sensitive subject matter (divorce, mental health, bullying, etc.)
  • Goals
    1. Promote reading aloud to young children
    2. Introduce children to these topics in a fun way OR help them understand and relate to what's happening around them

Functional Requirements (1-7 front end, 8-13 back end):

Browse All Books

FR-6: Users can view all books

FR-8: Add books to the Books table (in the BookFinder database)

More Info

FR-5: Users can view more information about a book

Browse by Topic

FR-2: Users can browse by topic

FR-10: Selecting books that match the chosen topic

Browse by Title/Author

FR-1: Users can search by title or author

FR-11: Selecting books that match a search keyword

Add/Remove Reading List Books

FR-3: Users can add a book to the Reading List

FR-4: Users can remove a book from the Reading List

FR-7: Users can view Reading List

FR-9: Marking books as “in the Reading List”

FR-12: Update the Books table by removing “deleted” Reading List books
